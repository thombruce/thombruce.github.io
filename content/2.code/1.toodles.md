# Toodles

_Toodles_ is an extended todo list format based on todo.txt.

Check out _Toodles_ for yourself here: [:icon{name="fa-brands:github"} thombruce/toodles](https://github.com/thombruce/toodles)

## Examples

### todo.txt

```todotxt
(A) 2024-07-20 Full @code +Todotxt #example tag:custom
x (A) 2024-07-21 2024-07-20 Full @code +Toodles #example tag:custom
```

### Toodles

```toodles
! (A) 2024-07-20 2024-07-31 £5.00-£10.00 Full @code +Toodles #example tag:custom x1
    ! (A) 2024-07-20 2024-07-31 £5.00-£10.00 Full @code +Toodles #example tag:custom x1
- (A) 2024-07-20 2024-07-31 £5.00-£10.00 Full @code +Toodles #example tag:custom x1
    - (A) 2024-07-20 2024-07-31 £5.00-£10.00 Full @code +Toodles #example tag:custom x1
(A) 2024-07-20 2024-07-31 £5.00-£10.00 Full @code +Toodles #example tag:custom x1
    (A) 2024-07-20 2024-07-31 £5.00-£10.00 Full @code +Toodles #example tag:custom x1
x (A) 2024-07-21 2024-07-20 2024-07-31 £5.00-£10.00 Full @code +Toodles #example tag:custom x1
    x (A) 2024-07-21 2024-07-20 2024-07-31 £5.00-£10.00 Full @code +Toodles #example tag:custom x1
~ (A) 2024-07-20 2024-07-31 £5.00-£10.00 Full @code +Toodles #example tag:custom x1
    ~ (A) 2024-07-20 2024-07-31 £5.00-£10.00 Full @code +Toodles #example tag:custom x1
```
